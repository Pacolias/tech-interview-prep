# Tech Interview Prep & Algorithms 🚀

This repository contains my optimized solutions and study notes for algorithmic and data structure problems, focusing primarily on the **LeetCode 75** list and **Database (SQL)** challenges.

## Objective
Intensive technical preparation for **Software Engineer (Backend / Distributed Systems)** and **Quantitative/Data Roles**.

## Technologies & Languages
- **Java:** Core implementation of object-oriented algorithms.
- **SQL:** Query optimization and database design.

## Methodology
The documented solutions prioritize:
1. **Time Complexity Reduction:** Seeking $O(n)$ or $O(\log n)$ solutions using advanced data structures (HashMaps, Heaps, Trees).
2. **Space Efficiency:** Smart exchanges between memory footprint and CPU time (Space-Time Tradeoffs).
3. **Clean Code:** Clear naming conventions and defensive handling of edge cases.
